# Static To-Do List App

This is a static stack To-Do List application.

This static to-do list app relies on fundamental web technologies—HTML, CSS, and JavaScript—to create a functional, local-first productivity tool, often acting as an entry-level project for understanding document object model (DOM) manipulation and browser-based data persistence. Unlike dynamic, server-side applications, a "static" approach often means the application runs entirely in the user's browser, utilizing localStorage to save tasks without needing a dedicated backend database.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Add tasks
- Delete tasks
- Tasks stored localStorage
